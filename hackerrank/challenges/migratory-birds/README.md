# Migratory Birds 🐦

## Platform

HackerRank

## Difficulty

Easy

## Language

Python 3


## Problem

Given an array of bird type IDs, find the type of bird that appears most frequently.

If multiple types have the same maximum frequency, return the smallest ID.


## My Approach

I used a Python dictionary to count the occurrences of each bird type.

Steps:

1. Create an empty dictionary
2. Iterate through the array
3. Store each bird frequency
4. Find the highest frequency
5. Return the smallest bird ID with this frequency


## Concepts Practiced

- Python dictionaries
- Frequency counting
- Lists traversal
- Conditional logic
- Problem solving


## Complexity Analysis

Time complexity: O(n)

Space complexity: O(k)

Where k is the number of different bird types.
